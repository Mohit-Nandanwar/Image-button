# Image with Button Project
This project demonstrates a simple webpage built using HTML and CSS.
It displays an image at the center of the page, and just below the image, there is a "Click Me" button.
# Features
- Clean layout using HTML structure.
- Styled image with CSS for proper alignment.
- A button placed below the image for interaction.
# How to Run
- Open the index.html file in any browser.
- You will see the image displayed with a "Click Me" button below it.
